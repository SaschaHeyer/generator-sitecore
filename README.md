## Getting Started

- Dependencies:
    - Node.js
    - Yeoman: `npm install -g yo`
- Install: `npm install -g generator-sitecore`
- Run: `yo sitecore`
- Update NuGet: `nuget restore`

> **NPM** https://www.npmjs.com/package/generator-sitecore
